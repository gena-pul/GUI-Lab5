# GUI Lab 5 – Quote Card Generator

## Overview

This project is a simple web application that generates a quote card with a random background image and a random quote.

When the user clicks the **Generate** button, the application:

1. Fetches a random quote from an API
2. Loads a random image from an image service
3. Combines them into a styled quote card
4. Displays the result with a smooth animation

The lab demonstrates **asynchronous JavaScript using `Promise.all()`** to run multiple tasks in parallel.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Promise.all()
